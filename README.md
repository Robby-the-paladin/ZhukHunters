# ZhukHunters
RK1 2022 (regularity checkers)

Вместо `dummy_regulator` вызываем в докере свой анализатор (предварительно нужно скачать туда зависимости и по надобности собрать).

Читаем грамматику строго из файла `test.CFG` в корне, пишем --- строго в файл `result` там же.

Если вердикт --- нерегулярность, то пишем `non-regular`, если регулярность --- пишем `regular`; если отказываемся решать либо прерываем решение --- пишем `unknown`.
Если возвращаете свидетельство (т.е. доказательство в какой-либо форме), то его пишем в отдельный файл с именем `proof_result` (ни в коем случае не в `witness.txt`, а то завесится программа-тестировщик).
